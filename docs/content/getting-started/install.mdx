Pick the path that matches your platform and how much you enjoy compiling things before coffee.

## Pre-built Binaries

Download the latest release directly from [GitHub Releases](https://github.com/kiro-generator/kiro-generator/releases/):

| Platform | Asset |
|---|---|
| Linux x86_64 | [kg-amd64-linux.tar.gz](https://github.com/kiro-generator/kiro-generator/releases/download/v0.2.2/kg-amd64-linux.tar.gz) |
| Linux arm64 | [kg-arm64-linux.tar.gz](https://github.com/kiro-generator/kiro-generator/releases/download/v0.2.2/kg-arm64-linux.tar.gz) |
| macOS arm64 | [kg-arm64-darwin.tar.gz](https://github.com/kiro-generator/kiro-generator/releases/download/v0.2.2/kg-arm64-darwin.tar.gz) |

Extract and place the binary somewhere on your `$PATH`:

```bash
tar -xzf kg-<platform>.tar.gz
sudo mv kg /usr/local/bin/
```

Verify checksums with [SHA256SUMS](https://github.com/kiro-generator/kiro-generator/releases/download/v0.2.2/SHA256SUMS).

## Homebrew (macOS)

`kiro-generator` is available through the `kiro-generator/kiro-generator` tap:

```bash
brew install kiro-generator/kiro-generator/kiro-generator
```

If you prefer the explicit two-step version:

```bash
brew tap kiro-generator/kiro-generator
brew install kiro-generator
```

Note: the current tap formula installs from the published `arm64-darwin` release asset.

## Arch Linux (AUR)

```bash
paru -S kiro-generator
```

If you use `yay`, that works too:

```bash
yay -S kiro-generator
```

## Debian/Ubuntu

Download the matching `.deb` from [GitHub Releases](https://github.com/kiro-generator/kiro-generator/releases/):

- [kiro-generator_0.2.2-1_amd64.deb](https://github.com/kiro-generator/kiro-generator/releases/download/v0.2.2/kiro-generator_0.2.2-1_amd64.deb)
- [kiro-generator_0.2.2-1_arm64.deb](https://github.com/kiro-generator/kiro-generator/releases/download/v0.2.2/kiro-generator_0.2.2-1_arm64.deb)

```bash
sudo apt install ./kiro-generator_0.2.2-1_amd64.deb
```

## Cargo

If you already have a Rust toolchain, this is the simplest cross-platform option:

```bash
cargo install kiro-generator
```

## From Source

```bash
git clone https://github.com/kiro-generator/kiro-generator.git
cd kiro-generator
cargo install --path .
```

## Verify

```bash
kg --version
kg --help
```
