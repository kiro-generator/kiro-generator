# Installation


## Cargo

```bash
cargo install kiro-generator
```

## Arch Linux AUR package

```bash
paru -S kiro-generator
```

```
git clone https://github.com/dougEfresh/kiro-generator.git
cd kiro-generator
cargo install --path .
```

```
kg --version
```
